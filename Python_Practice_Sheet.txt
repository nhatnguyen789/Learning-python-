# Embedded Test Engineer - Python Practice Sheet

## Mục tiêu:
Luyện tập các kỹ năng Python cơ bản đến trung cấp phục vụ cho kiểm thử hệ thống nhúng.

---
### 1. Basic Scripting
- Viết script in ra các số chia hết cho 5 từ 1 đến 100.
- Tính tổng các số từ 1 đến 1000 chia hết cho cả 3 và 7.

### 2. File Handling
- Viết script đọc 1 file `log.txt`, in ra tất cả các dòng chứa từ "ERROR".
- Đếm số dòng trong một file `.txt`.

### 3. Function & Modules
- Viết hàm nhận vào một chuỗi và trả về chuỗi đó viết ngược.
- Viết hàm nhận vào list số nguyên, trả về list mới chứa bình phương các số lẻ.

### 4. Dictionary & List
- Tạo dict từ 1 list các tuple (name, age). Tìm người lớn tuổi nhất.
- Từ list = [1,2,3,2,1,5], lọc ra các phần tử không lặp lại.

### 5. Test Automation (giả lập)
- Viết một hàm giả lập test case: nhận input là sensor_value, nếu lớn hơn 50 in ra "PASS", ngược lại "FAIL".
- Tạo danh sách 10 giá trị ngẫu nhiên (0-100) và kiểm thử với hàm trên.

### 6. Bonus: Time & Logging
- Viết script ghi log thời gian bắt đầu/kết thúc khi test script chạy.
